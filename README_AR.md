## 🚀 البدء السريع

### متطلبات أساسية
- Node.js 18+
- PostgreSQL 16
- Git
- GitHub و Render (للرفع)

### التثبيت (5 دقائق)

```bash
# 1. استنساخ
git clone [repo-url]
cd finaudit-platform

# 2. تثبيت المكتبات
npm install --legacy-peer-deps

# 3. إعداد قاعدة البيانات
# (تفاصيل في QUICK_START.md)

# 4. متغيرات البيئة
cp .env.example .env

# 5. التشغيل
cd apps/api && npm run dev      # Terminal 1 → 3001
cd apps/web && npm run dev      # Terminal 2 → 3000

# 6. افتح المتصفح
open http://localhost:3000
```

# FinAudit — Financial Compliance Platform

**An intelligent platform for financial compliance, risk management, and fraud detection**

---

## 🎯 Key Features

### 📊 Smart Dashboard
- Interactive KPIs (Revenue, VAT, Compliance %)
- Real-time charts and analytics
- Instant alerts for issues

### 🧠 Advanced AI
- **RPA**: Automatic invoice data extraction
- **Data Cleansing**: Financial data cleaning
- **Forecasting**: Revenue predictions
- **Fraud Detection**: 94% accuracy

### ✅ Automatic Compliance
- **ZATCA**: Invoice validation
- **Regulations**: Financial regulations tracking
- **Risk Management**: Interactive risk matrix

---

## 🚀 Quick Start

### Requirements
- Node.js 18+
- PostgreSQL 16
- Git
- GitHub & Render (for deployment)

### Installation (5 minutes)

